UnityTexturePaint possible to paint object in runtime.
By using Unity Texture Paint, you can make in various special directing becomes possible.


■ How to use

(1) Apply the DynamicCanvas component to the painting object
(2) Call DynamicCanvas from the script and call the DynamicCanvas.Paint method

Others We offer various functions useful for painting.
For details, please refer to [https://esprogram.github.io/UnityTexturePaintDocument/].
